# Smart Weather App 🌦️

This is a simple and modern weather web app that shows real-time weather information based on the user’s current location.

I built this project to practice JavaScript, API integration, and UI design using real-world data.

## What this app does
- Automatically detects your current location
- Shows live temperature, humidity, feels-like temperature
- Displays wind speed and air pressure
- Gives weather-based suggestions (rain, heat, cold, etc.)
- Shows a 6-day weather forecast
- Includes dark mode for better user experience

## Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## Why I built this project
I wanted to build a practical project that feels useful in daily life and helped me understand how APIs work and how to handle live data in JavaScript.

## How to run this project
1. Download or clone the repository  
2. Open the project folder  
3. Replace the API key in the JavaScript file with your own OpenWeatherMap API key  
4. Open `index.html` in any modern browser
